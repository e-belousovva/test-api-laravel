# test-api

How to install

- **git clone https://github.com/e-belousovva/test-api-laravel.git**
- **open the project**
- **cp env.example .env**
- **cd docker**
- **cp env.example .env**
- **docker-compose up -d**
- **docker-compose exec php-fpm bash**
- **composer install**
- **php artisan l5:generate**
